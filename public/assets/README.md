# Assets Directory

## Required Files

### Models
Place your 3D model files here:
- `models/f1_car.glb` - The F1 car model
- `models/driver.glb` - The driver model

### Audio (Optional)
Add audio files for enhanced experience:
- `audio/crowd.mp3` - Crowd ambience for intro
- `audio/engine.mp3` - Engine sound during driving
- `audio/finish.mp3` - Cheering/celebration at finish

### Resume
- `resume.pdf` - Your resume PDF file (already created as placeholder)

## Instructions

1. Copy your `.glb` model files to `public/assets/models/`
2. (Optional) Add audio files to `public/assets/audio/`
3. Replace `resume.pdf` with your actual resume
